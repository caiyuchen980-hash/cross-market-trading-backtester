# cross-market-trading-backtester
A Python-based quantitative backtesting system for comparing five trading strategies (Buy & Hold, MACD, RSI, MA Envelope, MA Oscillator) across US, Hong Kong, and China A-share markets. Includes real-time data scraping, options sentiment analysis, and strategy performance visualization.

## ✨ Features
- Real-time stock data scraping via the yfinance API
- Options sentiment analysis (open interest, implied volatility, max pain price)
- Backtesting of five common trading strategies:
  - Buy & Hold (benchmark)
  - MACD
  - RSI
  - Moving Average Envelope
  - Moving Average Oscillator
- Cross-market performance comparison across US, Hong Kong, China A-share markets
- Visualization of equity curves and performance metrics
